# Getting Started

Welcome to the **Online Saathi Identity Provider (IDP)** documentation. Online Saathi provides a secure, standards-compliant (OAuth 2.0 & OIDC) Single Sign-On (SSO) solution for your applications.

## Features

- **Single Sign-On (SSO)**: One login for all your applications.
- **Secure Authentication**: Built on industry-standard OAuth 2.0 and OpenID Connect protocols.
- **Developer Friendly**: Easy integration with simple REST APIs and standard libraries.
- **User Dashboard**: Centralized profile and security management for users.

## Quick Start

### 1. Create an Account
If you haven't already, [Sign Up](https://accounts.i-sewa.in/register) for an Online Saathi account.

### 2. Register Your Application
To integrate your app with Online Saathi, you need to obtain a **Client ID** and **Client Secret**.

*Currently, application registration is handled manually by the administrator. Please contact support or the admin to provision your application.*

### 3. Integrate Authentication
Once you have your credentials, you can integrate Online Saathi using our [OAuth Integration Guide](./oauth-integration.md).

## Support

If you encounter any issues, please reach out to our support team or check the API Reference.
